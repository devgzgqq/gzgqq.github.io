Typeface 字体

跨平台的字体设定，力求在各个操作系统下都有最佳展示效果。

字体家族

  • 中文字体

  • 英文字体

字体家族 css 代码如下：

font-family: "Helvetica Neue",Helvetica,"PingFang SC","Hiragino Sans GB","Microsoft YaHei","微软雅黑",Arial,sans-serif;

文本字体颜色

通过颜色来展示意图，这些类可以应用于链接，并且在鼠标经过时颜色可以还可以加深，就像默认的链接一样。

基础文本颜色 #333 如 标题

基础文本颜色 #555 如 文本和连接颜色，

辅助文本颜色 #999 如 辅助信息

首选文本颜色 #28a3ef 如 首选文本颜色

成功文本颜色 #87d068 如 提示输入正确

错误文本颜色 #ea4a36 如 提示输入错误

告警文本颜色 #fac603 如 告警提示
